# Synthetizer
3 Octaves synthetizer piano made of PCM5100, LM4810 and PAM8403 
## Features
- Volume control from the screen
## BOM
- 1× Orpheus_Pico (Raspberry Pi Pico)
- 5× Capacitor 1 µF
- 4× Capacitor 4.7 µF
- 1× Capacitor 0.1 µF
- 1× Capacitor 0.1 µF
- 2× Capacitor 10 µF
- 41× Diodo 1N4148
- 4× Mounting Hole M3
- 1× Pantalla ST7735R LCD
- 1× Jack de audio 3.5 mm, 4 polos (AudioJack4_SwitchT1)
- 2× Pad de conexión (+)
- 2× Pad de conexión (–)
- 2× Resistencia 10 kΩ
- 40× Botón pulsador (SW_Push)
- 1× Rotary Encoder con switch (RotaryEncoder_Switch_MP)
- 1× DAC PCM5100
- 1× LM4810
- 1x PAM8403
- 4x M3 Heatsets
- 4x M3x16 screws
