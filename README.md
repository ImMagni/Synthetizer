# Synthetizer
<img src=images/3dviewpcb.png/> <br>
3 Octaves synthetizer piano made of PCM5100, LM4810 and PAM8403 
## Features
- Volume control from the screen
- Midi, Speakers, and line output.
- 3 octaves
- four types of waves to make audio
## BOM
- 1× Orpheus_Pico (Raspberry Pi Pico)
- 5× Capacitor 1 µF
- 4× Capacitor 4.7 µF
- 1× Capacitor 0.1 µF
- 1× Capacitor 0.1 µF
- 2× Capacitor 10 µF
- 41× Diode 1N4148
- 1× screen ST7735R LCD
- 1× Jack de audio 3.5 mm
- 2× Resistencia 10 kΩ
- 40× Botón pulsador
- 1× Rotary Encoder with switch
- 1× DAC PCM5100
- 1× LM4810
- 1x PAM8403
- 5x M3 Heatsets
- 5x M3x16 screws
